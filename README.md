# Employee-Tracker

## Author: Nayandeep Sidhu 

### Description

Employee Tracker allows an Admin user to View Departments, Employees and Roles that are set within the company. The application also allows the user to view salaries of each role and view Job Designations of employees across the board. The application enables the user to Add new Departments, Add new Employees, Add new Roles and assign Managers to new Employees. The application also allows the user to update an existing employee profile. When finished with the application, user can Quit the application by selecting the Quit option.
